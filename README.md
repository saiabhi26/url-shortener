# URL Shortener

A URL shortener built with FastAPI + SQLAlchemy, built step by step as a system design learning project.

Short codes are generated by base62-encoding the auto-increment primary key —
no separate short_code column, no random collisions to worry about.

### Setup

```bash
python3 -m venv .venv
source .venv/bin/activate   # on Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### Run

```bash
uvicorn app.main:app --reload
```

App will be available at http://localhost:8000
Interactive API at http://localhost:8000/docs 

### Try it

```bash
curl -X POST http://localhost:8000/shorten \
  -H "Content-Type: application/json" \
  -d '{"url": "https://www.example.com/some/very/long/path"}'
```

Then visit the returned `short_url` in your browser, or curl it (with `-L` to follow the redirect).
